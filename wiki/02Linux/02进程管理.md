### 查看进程

##### 1. ps

```
# 查看自己的进程,仅用户进程
# ps [-l/m]
# 查看系统所有进程,包括root进程
# ps aux
# 查看特定的进程
# ps aux | grep [进程名字]
```

##### 2. pstree

```
# 查看所有进程树
# pstree -A
```

##### 3. top

```
# 实时显示进程信息,两秒刷新一次
# top -d [number]
```

##### 4. netstat

```
# 查看特定端口的进程
# netstat -anp | grep port
```
